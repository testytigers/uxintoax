## The Short Answer

AI agent personas are the personality, voice, and behavior profiles of AI agents. Design them by defining purpose, choosing personality traits, specifying communication style, setting response patterns, and testing with users. The key is alignment — the persona should match both your brand and the agent's purpose.

---

## The Core Framework

### 1. Define Purpose

What is the agent trying to achieve? Purpose drives personality.

### 2. Choose Personality Traits

Select 3-5 personality traits that define the agent.

**Options:**
- Friendly, Professional, Playful, Serious
- Direct, Conversational, Instructional
- Confident, Humble, Cautious

### 3. Specify Communication Style

How does the agent communicate?

**Elements:**
- Tone (formal, casual)
- Length (concise, detailed)
- Style (direct, conversational)
- Vocabulary (technical, simple)

### 4. Set Response Patterns

How does the agent respond in different situations?

**Patterns:**
- Standard requests
- Error responses
- Uncertainty responses
- Follow-up questions

### 5. Test with Users

Test the persona with real users to ensure it feels natural and appropriate.

---

## The Bottom Line

AI agent personas define how agents communicate. Design them by aligning personality, tone, and style with the agent's purpose and your brand. Test with users to ensure the persona feels natural.
