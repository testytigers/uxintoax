## The Short Answer

These 8 AX design projects showcase the skills employers are looking for in 2025. Each project demonstrates a different aspect of AI-era UX design — from agent design to trust patterns to adaptive interfaces.

---

## 1. AI Chatbot UX

Design the complete UX for an AI-powered chatbot that helps users accomplish a complex task — booking travel, filing taxes, or managing finances. Include conversation design, error handling, and confidence indicators.

**Skills shown:** Conversational design, error recovery, trust patterns, content generation

---

## 2. AI Agent Dashboard

Design a dashboard for managing AI agents — monitoring agent behavior, reviewing agent decisions, and intervening when agents make mistakes. Include real-time monitoring, decision logs, and intervention flows.

**Skills shown:** System design, monitoring UX, intervention patterns, data visualization

---

## 3. Personalized Learning Platform

Design an AI-powered learning platform that adapts content and pacing based on user progress. Include personalization settings, progress tracking, and adaptive content delivery.

**Skills shown:** Adaptive design, personalization UX, progress tracking, content strategy

---

## 4. AI Writing Assistant

Design the UX for an AI writing tool that helps users write emails, reports, or documentation. Include drafting, editing, tone adjustment, and explanation of AI suggestions.

**Skills shown:** Generative UI, explanation patterns, tone controls, editing UX

---

## 5. AI-Powered Research Tool

Design a UX for a research tool that uses AI to analyze user data, generate insights, and recommend next steps. Include data visualization, insight presentation, and recommendation flows.

**Skills shown:** Data visualization, insight presentation, recommendation UX, analysis design

---

## 6. Voice-First Interface

Design a voice-first interface for a common task — ordering food, checking weather, or managing calendar events. Include voice interaction design, error recovery, and visual fallback.

**Skills shown:** Voice UX, multimodal design, error recovery, visual fallback design

---

## 7. AI Image Generation Tool

Design the UX for an AI image generation tool that helps non-designers create images from text prompts. Include prompt input, style selection, iteration controls, and image review flows.

**Skills shown:** Generative interface, prompt design, iteration UX, visual feedback

---

## 8. AI Customer Service Interface

Design the UX for an AI-powered customer service interface that handles customer inquiries, escalates to humans when needed, and learns from interactions. Include conversation design, escalation patterns, and knowledge base integration.

**Skills shown:** Conversational design, escalation UX, knowledge management, human-in-the-loop design

---

## The Bottom Line

Build 3-5 of these projects and present them as case studies. Focus on your process, not just the output. Show how you approach AI-era design problems, how you handle uncertainty, and how you design for trust and transparency.
